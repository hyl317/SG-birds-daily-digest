# SG Birds Daily Summary

A Python script that reads the "SG BIRDS (Sightings/Live Update)" Telegram group, summarizes bird sightings using Claude Haiku, and sends the digest to the user's Telegram Saved Messages. Scheduled via macOS launchd.

## Architecture

Single-file Python script (`sg_birds_summary.py`) with three main stages:
1. **Fetch** — Telethon reads messages from the configured Telegram group within a time window
2. **Summarize** — Claude Haiku extracts bird species, locations, and notable details
3. **Send** — Summary is delivered to Telegram Saved Messages as an HTML-formatted message

## Key Files

- `sg_birds_summary.py` — Main script (fetch, summarize, send)
- `acronyms.md` — Known acronyms for bird species, locations, and terms. Claude uses this to expand abbreviations. New acronyms are auto-detected and appended by the script.
- `config.json` — Runtime config (group ID, schedule). Generated by interactive setup on first run.
- `group_config.json` — Static group metadata
- `.env` — API keys (Telegram + Anthropic). Never commit.
- `com.hyl.sgbirds-summary.plist` — launchd schedule definition (auto-generated by setup)
- `session/` — Telethon session file (persists Telegram login)

## Dependencies

- `telethon` — Telegram client
- `anthropic` — Claude API (uses `claude-haiku-4-5-20251001`)
- `python-dotenv` — Env file loading

## Commands

```bash
# Install dependencies
pip install -r requirements.txt

# First run (interactive setup: select group, time, frequency)
python sg_birds_summary.py

# Re-configure (delete config to re-trigger setup)
rm config.json && python sg_birds_summary.py

# Check launchd status
launchctl list | grep sgbirds

# View logs
cat /tmp/sg_birds_summary.log
```

## Notes

- The script is designed to run unattended via launchd. First run must be manual for Telegram login.
- `acronyms.md` is both human-editable and machine-appended. Manual edits (corrections, removals) take precedence — the script only appends new entries.
- The summarization prompt explicitly instructs Claude to only report species named in text, not infer from photos.
- Messages are capped at 600K chars before sending to Claude. Haiku 4.5 has a 200K token context window (~800K chars); the 600K limit leaves headroom for the prompt itself.
